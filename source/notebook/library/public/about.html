About Notebooks

The concept of this library is inspired by the Notebook library from Rod Takehara.

It is has been built from the ground up though, utilitizing functionality from the latest version of MapTool, as of this writing 1.14.

In addition it serves as a show-case for using javascript and modern frameworks for creating reactive web applications, but in MapTool.
